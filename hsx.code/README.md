# HSX Code

HSX Code is a next-generation code editor designed for **.hsx** and **.ks** languages with full **extension support**, custom runtime environments, and a polished Electron + React frontend.

---

## Features

- Advanced editor with **Monaco integration**
- Support for **.hsx** and **.ks** languages
- Real-time **syntax highlighting**, **linting**, **autocomplete**, **snippets**
- **Extensions marketplace** (upload, sandboxed execution)
- Fully **customizable settings**
- Multi-platform support (**Windows, macOS, Linux**)
- Integrated **HSX runtime** (JavaScript + Python via Pyodide)
